## Setup

1. Git clone this repo
2. [Install Node.js / npm](http://nodejs.org/download/)
3. [Install PhantomJS](http://phantomjs.org/)
3. Fetch the dependencies with npm:

        npm update

4. Install Gulp

        npm install -g gulp

5. Verify Gulp works

        gulp
